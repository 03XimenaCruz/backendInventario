## Instalación y Configuración

### Clonar el repositorio

#### git clone https://github.com/usuario/tu-repo.git

### Entrar al proyecto
#### cd nombre-proyecto

### Instalar dependencias
#### npm install

### Crear archivo .env
#### DB_HOST=localhost
#### DB_USER=root
#### DB_PASSWORD=tu_password
#### DB_NAME=tu_bd
#### PORT=3000

### Ejecutar proyecto

#### npm run dev (desarrollo)
#### npm start (producción)
